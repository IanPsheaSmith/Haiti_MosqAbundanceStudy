Haiti Mosquito Abundance Study

This repository contains data, code, predictions, and figures associated with a spatial modeling study of mosquito abundance in Haiti. The study aims to understand the ecological and environmental drivers of six mosquito species based on species-specific environmental buffers and spatiotemporal covariates.

Data and Code

View Data_and_Code Folder

Mosquito Datasets (6 CSV files):

Each species-specific dataset includes date, coordinates, mosquito counts, and environmental covariates extracted using buffer zones tailored to each species' dispersal range.

Available files:

  • Species datasets (temperature, precipitation, NDVI, population density, elevation, etc.)
  • R Markdown Analysis Code
  • Download Shapefiles (.zip)

Raster Data

Contains environmental covariate raster stacks used in the analysis:

  • Monthly raster stacks for temporal analysis
  • Average monthly covariates for baseline predictions

Species Covered

Aedes aegypti (Aeae)
Aedes albopictus (Aealb)
Aedes mediovittatus (Aem)
Culex quinquefasciatus (Quinx)
Culex nigripalpus (Cxn)
Psorophora columbiae (Psc)

Usage Instructions

  1. Clone the repository: 
    git clone https://github.com/IanPsheaSmith/Haiti_MosqAbundanceStudy.git
  2. Open Haiti_MosqAbundance.Rmd in RStudio
  3. Install required R packages (listed at the top of the .Rmd file)
  4. Knit the document to reproduce model outputs and figures
